## [0.1.0] - 2026-07-17

### Added

- OTLP JSON and JSONL span normalization.
- GenAI semantic-convention model, token, operation, and tool extraction.
- Exclusive-time critical path, tool error, and retry-loop analysis.
- User-configurable exact, prefix, and wildcard token pricing.
- Rich terminal summaries plus stable JSON and self-contained HTML reports.
- Built-in demo, realistic examples, typed API, tests, and multi-version CI.
